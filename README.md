CookingAnEgg
============

## PHP version
```bash
cd php
```

### Run with PHP
```bash
composer install
php home.php
```

### Run without PHP
```bash
./build.sh
./run.sh
```


## F# version
```bash
cd F#
```

### Run with fsharpi
```bash
fsharpi home.fsx
```
